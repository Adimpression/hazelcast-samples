hazelcast-samples
=================

This repository contains simple samples related to Hazelcast usage:
* in-memory distributed cache
* write-through distributed cache
* simple MPP via executor service

Sample application is created using:
* Spring Boot 1.0.1
* Hazelcast 3.2
* Postgres 9.3
